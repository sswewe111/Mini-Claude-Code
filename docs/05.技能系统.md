# 技能系统

***不是把所有知识永远塞进 prompt，而是在需要的时候再加载正确那一份。***

**把“长期可选知识”从 system prompt 主体里拆出来，改成按需加载。**

```python
第 1 层：轻量目录
  - skill 名称
  - skill 描述
  - 让模型知道“有哪些可用”

第 2 层：按需正文
  - 只有模型真正需要时才加载
  - 通过工具结果注入当前上下文
```

## 1、定义技能信息摘要数据类和完整技能文档数据类

技能信息摘要

name：技能名称

description：技能信息描述，用于模型判断是否使用这份技能

path：技能文件路径，用于读取完整的技能文档

```python
@dataclass
class SkillManifest:
    name: str
    description: str
    path: Path
```

完整技能文档

manifest：技能摘要信息，包括名字、描述、路径

body：技能正文，也就是 `SKILL.md` 中真正的完整指导内容

```python
@dataclass
class SkillDocument:
    manifest: SkillManifest
    body: str
```

## 2、制定技能注册表

_load_all()： 加载所有技能，并封装成SkillDocument

_parse_frontmatter()：解析 Markdown 文件头配置

describe_available()：生成技能目录说明

load_full_text()：按名称加载完整技能内容

```python
class SkillRegistry:
    def __init__(self, skills_dir: Path):
		    #保存技能目录
        self.skills_dir = skills_dir
        #创建一个空字典，用于保存所有技能文档
        self.documents: dict[str, SkillDocument] = {}
        #调用 _load_all() 加载所有技能
        self._load_all()
    
    # 加载所有技能，并封装成SkillDocument    
    def _load_all(self) -> None:
        if not self.skills_dir.exists():
            return
        for path in sorted(self.skills_dir.rglob("SKILL.md")):
		        #meta里包含name和description
            meta, body = self._parse_frontmatter(path.read_text())
            name = meta.get("name", path.parent.name)
            description = meta.get("description", "No description")
            manifest = SkillManifest(name=name, description=description, path=path)
            self.documents[name] = SkillDocument(manifest=manifest, body=body.strip())
    
    #解析 Markdown 文件头配置        
    def _parse_frontmatter(self, text: str) -> tuple[dict, str]:
        match = re.match(r"^---\n(.*?)\n---\n(.*)", text, re.DOTALL)
        if not match:
            return {}, text
        meta = {}
        for line in match.group(1).strip().splitlines():
            if ":" not in line:
                continue
            key, value = line.split(":", 1)
            meta[key.strip()] = value.strip()
        return meta, match.group(2)
    
    #生成技能目录说明[{name:description},{name:description}]    
    def describe_available(self) -> str:
        if not self.documents:
            return "(no skills available)"
        lines = []
        for name in sorted(self.documents):
            manifest = self.documents[name].manifest
            lines.append(f"- {manifest.name}: {manifest.description}")
        return "\n".join(lines)
        
    def load_full_text(self, name: str) -> str:
        document = self.documents.get(name)
        if not document:
            known = ", ".join(sorted(self.documents)) or "(none)"
            return f"Error: Unknown skill '{name}'. Available skills: {known}"
        return (
            f"<skill name=\"{document.manifest.name}\">\n"
            f"{document.body}\n"
            "</skill>"
        )
        

SKILL_REGISTRY = SkillRegistry(SKILLS_DIR)
```

md文件格式

`match = re.match(r"^---\n(.*?)\n---\n(.*)", text, re.DOTALL)`解析md文件

`match.group(1)`：表示 frontmatter 配置部分。即name和description

`match.group(2)` ：表示正文部分。

```markdown
---
name: xxx
description: xxx
---

正文内容
```

## 5、模型使用技能工具

```python
def agent_loop(messages: list) -> None:
    while True:
        response = client.messages.create(
            model=MODEL,
            system=SYSTEM,
            messages=messages,
            tools=TOOLS,
            max_tokens=8000,
        )
        messages.append({"role": "assistant", "content": response.content})
        if response.stop_reason != "tool_use":
            return
        results = []
        for block in response.content:
            if block.type != "tool_use":
                continue
            handler = TOOL_HANDLERS.get(block.name)
            try:
                output = handler(**block.input) if handler else f"Unknown tool: {block.name}"
            except Exception as exc:
                output = f"Error: {exc}"
            print(f"> {block.name}: {str(output)[:200]}")
            results.append({
                "type": "tool_result",
                "tool_use_id": block.id,
                "content": str(output),
            })
        messages.append({"role": "user", "content": results})
```

## 总体流程

```python
程序启动
  ↓
加载 .env 环境变量
  ↓
确定当前工作目录 WORKDIR
  ↓
扫描 skills/ 目录下的所有 SKILL.md
  ↓
提取每个技能的 name 和 description
  ↓
把技能目录放进 SYSTEM 提示词
  ↓
等待用户输入
  ↓
模型根据用户问题决定是否调用工具
  ↓
如果需要专业技能：
      调用 load_skill
      加载完整 SKILL.md 正文
      再根据技能内容执行任务
  ↓
如果需要读写文件或执行命令：
      调用 read_file / write_file / edit_file / bash
  ↓
模型生成最终回答
```
